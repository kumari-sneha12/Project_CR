# 📌 About the Project

The **Car Rental Website** is a web-based application developed using **Python, Django, MySQL, HTML, CSS, JavaScript, and Bootstrap**. It allows users to search, compare, and book rental cars online while providing administrators with a dashboard to manage vehicles, customers, bookings, and payments.

This project was developed as a **Bachelor of Computer Applications (BCA) Final Year Project**.

# 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/your-username/car-rental-website.git
```

Go to Project

```bash
cd car-rental-website
```

Create Virtual Environment

```bash
python -m venv venv
```

Activate Virtual Environment

Windows

```bash
venv\Scripts\activate
```

Linux / Mac

```bash
source venv/bin/activate
```

Install Dependencies

```bash
pip install -r requirements.txt
```

Run Migrations

```bash
python manage.py makemigrations
```

```bash
python manage.py migrate
```

Create Superuser

```bash
python manage.py createsuperuser
```

Run Server

```bash
python manage.py runserver
```

Open Browser

```
http://127.0.0.1:8000/
```
